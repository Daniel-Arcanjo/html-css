A Melhor Maneira

O estilo das CSS externa é a melhor forma de aplicá-las, para aplicar, usa a tag "link:css" e tudo que estiver nela será aplicada a todos as páginas de um site


Para fins de comparação, o css local é extremamente pontual, o interno é mais abrangente e o externo o principal. É possível misturar todos em um só código.
